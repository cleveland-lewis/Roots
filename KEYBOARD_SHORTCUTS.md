## Calendar Navigation

### Date Navigation
| Shortcut | Action |
|----------|--------|
| ←→ | Previous/Next Day |
| ↑↓ | Previous/Next Week |
| ⌘← | Previous Day (Global) |
| ⌘→ | Next Day (Global) |
| ⌘⌥← | Previous Week (Global) |
| ⌘⌥→ | Next Week (Global) |
| ⌘T | Go to Today |

### Calendar Grid (When Focused)
| Shortcut | Action |
|----------|--------|
| ← | Move selection left (previous day) |
| → | Move selection right (next day) |
| ↑ | Move selection up (previous week) |
| ↓ | Move selection down (next week) |
| Return | Select highlighted date |
| Space | Select highlighted date |

**Tip**: The calendar grid automatically receives focus when you open the Calendar page. Just start pressing arrow keys!

## Accessibility

### VoiceOver
All keyboard shortcuts work with VoiceOver enabled. Focus changes are announced automatically.

### Full Keyboard Access
Enable in **System Settings → Keyboard → Keyboard Navigation** to navigate all UI elements with keyboard.

## Tips & Tricks

### Efficient Calendar Navigation
1. Press ⌘T to jump to today
2. Use ← → to browse days
3. Press ↑ ↓ to jump weeks
4. Press Return to select and view events

### Quick Event Creation
1. Press ⌘N anywhere in the app
2. Event sheet opens
3. Fill details
4. Press ⌘Return to save

### Focus Mode Workflow
1. Press ⌘⌥F to enter focus mode
2. Minimize distractions
3. Press ⌘⌥F again to exit

## Troubleshooting

### Shortcuts Not Working?

**Check**:
1. Full Keyboard Access enabled
2. No conflicting shortcuts in System Settings
3. App has focus (click window if needed)
4. Not in text input field

**Common Issues**:
- **Arrow keys don't work**: Make sure calendar grid has focus
- **⌘N doesn't work**: Make sure you're not in a text field
- **Tab doesn't work**: Enable Full Keyboard Access

### Accessibility
If using VoiceOver or other assistive tech:
- All shortcuts remain functional
- Focus changes are announced
- Use VO+Space to activate buttons
